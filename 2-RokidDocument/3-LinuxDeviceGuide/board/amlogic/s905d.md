## 编译指令

目前仅支持64位版本。
编译S905D请选择'nana_t2_s905d_release'

```
source rokid_br_external/build/setenv.sh
```
输出
```
Environment setting is OK!
Just type 'lunch' and you will get a list of choices, or you can type 'lunch [choice]' to lunch directly.
```

lunch
